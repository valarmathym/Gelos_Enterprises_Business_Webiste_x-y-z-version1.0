# Gelos Enterprise Business Website – Version 1.0
### Live Demo: https://valarmathym.github.io/Gelos_Enterprises_Business_Webiste_x-y-z-version1.0/

## Project Overview
Gelos Enterprise Business Website v1.0 is a responsive and accessible business website developed for Gelos Enterprises, a technology services company. The website showcases the company’s services, values, and case studies through a modern, user-friendly, and interactive interface.

The project follows the official Gelos style guide to ensure consistency in branding, typography, color schemes, spacing, and user interface design.

---

## Features

### 1. Case Studies Section
- Interactive image slider/carousel
- Swipe functionality with slide indicators
- Background images with call-to-action buttons
- Links to dedicated case study pages

### 2. Company Offerings
- Accordion-style expandable content
- Service categories:
  - Digital Transformation
  - Cloud Services
  - Artificial Intelligence
- Internal navigation links

### 3. Company Values
- Grid-based responsive layout
- Core values displayed:
  - Integrity
  - Inclusivity
  - Innovation
- Icons with descriptive content

### 4. Contact Us Form
- Split-layout contact section
- Real-time form validation
- Input restrictions and error handling:
  - Name validation
  - Phone number validation
  - Message character limits
- Required field checks before submission
